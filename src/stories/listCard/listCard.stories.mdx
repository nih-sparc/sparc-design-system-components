# List Card
List cards are used to display a link to another website or web page that provides additional information about a single subject.
They contain a brief overview of the resource, date, link to the website or webpage, and an image of the topic, if available.
They also contain an optional section text at the bottom that links the user to another webpage.

# List Card

## Properties
The List Card component accepts the following properties:

`data`: The data associated with the list card items. Array. Required. Each data entry must have a title, summary, url, and date property assigned. 
They can also have an optional image property set, which will display an image to the left.

`sectionText`: The text that will be shown below the list card items and used for navigation purposes. String. Optional.

`sectionUrl`: The url to naviagte to when a user clicks on the section text. String. Optional.
