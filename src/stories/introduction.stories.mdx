# Introduction

This design system serves a variety of audiences, and is available in both Abstract format (a repository for designer files) to Developers/Engineers and Designers, and the SPARC Design System Git-Hub Storybook (https://nih-sparc.github.io/sparc-design-system-components/) to use approved recommendations and components. Design System Recommendations in Process can be found in Abstract.
The Built Design System components are available in the following Git-Hub Repository https://github.com/nih-sparc/sparc-design-system-components
