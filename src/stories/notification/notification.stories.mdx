# Notifications

### Information Notification

<Story name="Information Notification">{stories.InformationNotification()}</Story>

#### Code

```js
import { informationNotification } from "@/utils/notificationMessages"

data() {
  return {
    notification : informationNotification('Notification Title', 'This is a notification.')
  }
},
methods() {
  clickHandler() {
    this.$notify(this.notification)
  }
}
```

```html
<el-button
  plain
  @click="clickHandler">
  Show Notification
</el-button>
```
