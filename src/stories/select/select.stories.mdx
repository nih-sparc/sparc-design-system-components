# Select
Select inputs can be used when it is desired to display several options and allow the user to select one. 
We directly utilize the Element UI el-select component for this which requires a value to bind to using `v-model`.

<Story name="Normal">{stories.Primary()}</Story>

#### Code
```html
<el-select v-model="value">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```

```js
export default {
  data() {
    return {
      options: [{
        value: 'Option1',
        label: 'Option 1'
      }, {
        value: 'Option2',
        label: 'Option 2'
      }, {
        value: 'Option3',
        label: 'Option 3'
      }, {
        value: 'Option4',
        label: 'Option 4'
      }, {
        value: 'Option5',
        label: 'Option 5'
      }],
      value: ''
    }
  }
}
```

# Grouping
Options can be categorized in groups and displayed with a label for each group.

<Story name="Grouping">{stories.Grouping()}</Story>

#### Code
```html
<el-select v-model="value">
  <el-option-group
  v-for="group in options"
  :key="group.label"
  :label="group.label"
  >
    <el-option
      v-for="item in group.options"
      :key="item.value"
      :label="item.label"
      :value="item.value">
    </el-option>
  </el-option-group>
</el-select>
```

```js
export default {
  data() {
    return {
      options: [
        {
          label: 'Group 1',
          options: [
            {
              value: 'Option1',
              label: 'Option 1'
            },
            {
              value: 'Option2',
              label: 'Option 2'
            },
          ]
        },
        {
          label: 'Group 2',
          options: [
            {
              value: 'Option3',
              label: 'Option 3'
            },
            {
              value: 'Option4',
              label: 'Option 4'
            },
          ]
        },
      ],
      value: []
    }
  }
}
```
