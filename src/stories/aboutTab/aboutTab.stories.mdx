# About Tabs
Tabs are used to quickly navigate between views within the same context. The views are used to group similar information on the same page.

## Properties
The About Tab component accepts the following properties:

`tabs`: The tabs associated with the components. Array. Required.

`activeTab`: The currently active tab. String. Optional.

### About Tab Style One

<Story name="Style One">{stories.AboutTabStyleOne()}</Story>

#### Code

```js
data() {
  return {
    activeTab: 'organs',
    tabs: [
      {
        label: 'Datasets',
        id: 'datasets'
      },
      {
        label: 'Organs',
        id: 'organs'
      },
      {
        label: 'Images',
        id: 'images'
      },
      {
        label: 'Projects',
        id: 'projects'
      },
      {
        label: 'Simulations',
        id: 'simulations'
      }
    ]
  }
}
```

```html
<tab-nav
  :tabs="tabs"
  :active-tab="activeTab"
  @tab-changed="activeTab = $event"
/>
```
