# About Tabs
About tabs are used to quickly navigate between views within the same context. The views are used to group similar information on the same page.

About tabs make use of the content tab card, but leave the available data slot empty.

## Properties
The About Tab component accepts the following properties:

`tabs`: The tabs associated with the components. Array. Required.

`activeTabId`: The id of the active tab. String. Required.

## Events
`tab-changed`: Fire when the user selects a new tab. Returns the new selected tab.

### About Tab Style One

<Story name="Style One">{stories.AboutTabStyleOne()}</Story>

#### Code

```js
data() {
  return {
    activeTabId: 'organs',
    tabs: [
      {
        label: 'Datasets',
        id: 'datasets'
      },
      {
        label: 'Organs',
        id: 'organs'
      },
      {
        label: 'Images',
        id: 'images'
      },
      {
        label: 'Projects',
        id: 'projects'
      },
      {
        label: 'Simulations',
        id: 'simulations'
      }
    ]
  }
},
methods: {
  tabChanged(newTab) {
    this.activeTabId = newTab.id
  }
}
```

```html
<content-tab-card
  :tabs="tabs"
  :active-tab-id="activeTabId"
  @tab-changed="tabChanged"
/>
```
