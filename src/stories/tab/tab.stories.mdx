# Tabs
Tabs are used to quickly navigate between views within the same context. The views are used to group similar information on the same page.

Tabs make use of the content tab card, but leave the available data slot empty.

## Properties
The Tab component accepts the following properties:

`tabs`: The tabs associated with the components. Array. Required.

`activeTabId`: The id of the active tab. String. Required.

`tabStyle`: The type of style that you want to be applied. String. Optional. The options are: 'style1', 'style2', and 'style3'. Defaults to 'style1'

## Events
`tab-changed`: Fire when the user selects a new tab. Returns the new selected tab.

### Tab Style One

<Story name="Style One">{stories.TabStyleOne()}</Story>

#### Code

```js
data() {
  return {
    activeTabId: 'organs',
    tabs: [
      {
        label: 'Datasets',
        id: 'datasets'
      },
      {
        label: 'Organs',
        id: 'organs'
      },
      {
        label: 'Images',
        id: 'images'
      },
      {
        label: 'Projects',
        id: 'projects'
      },
      {
        label: 'Simulations',
        id: 'simulations'
      }
    ]
  }
},
methods: {
  tabChanged(newTab) {
    this.activeTabId = newTab.id
  }
}
```

```html
<content-tab-card
  :tabs="tabs"
  :active-tab-id="activeTabId"
  @tab-changed="tabChanged"
/>
```
