# MultiSelect

## Properties
The radio button component accepts the following properties:

`options`: The options associated with the multiselect. String, Number, or Boolean. Required. In order to make the select multi-level then children must be added to ALL the parents. For single level, NO children should be added to ANY of the data.

### Single Level MultiSelect

<Story name="Single Level MultiSelect">{stories.SingleLevel()}</Story>

#### Code

```js
data() {
  return {
    options: [
    {
      value: 1,
      label: 'Asia',
    }, {
      value: 14,
      label: 'Europe',
    }, {
      value: 23,
      label: 'North America',
    }]
  }
}
```

```html
<multi-select
  :options="options"
/>
```
