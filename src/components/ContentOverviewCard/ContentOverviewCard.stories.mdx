# Content Overview Card

Primarily used for overview pages for datasets, organs, projects, and simulations.

## Properties
The Content Overview Card component accepts the following properties:

`title`: The title associated with the component. String. Required.

`image`: The image associated with the component. String. Required.

`description`: The description associated with the component. String. Required.

`subtitle`: The subtitle associated with the component. String. Optional.

Content Overview Card contains named 'metadata' and 'buttons' slots, both of which are optional.
